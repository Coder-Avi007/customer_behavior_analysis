# Customer Shopping Behavior Analysis

## 📌 Overview

This project analyzes customer shopping behavior using Python, SQL, and Power BI to uncover insights into customer demographics, purchasing patterns, product performance, and subscription trends.

The workflow covers the complete data analytics lifecycle, including data cleaning, exploratory data analysis (EDA), database querying, dashboard creation, reporting, and presentation development.

---

## 📂 Dataset

The dataset contains customer transaction records and shopping behavior information, including:

* Customer demographics
* Product categories
* Purchase amounts
* Subscription status
* Discount usage
* Shipping preferences
* Review ratings
* Purchase frequency

### Dataset Size

* Rows: 3,900
* Columns: 18

## 🚀 How to Run

### Clone Repository

```bash
git clone https://github.com/Coder-Avi007/customer-shopping-behavior-analysis.git
```

### Install Dependencies

```bash
pip install pandas numpy matplotlib seaborn sqlalchemy psycopg2
```

### Run Jupyter Notebook

```bash
jupyter notebook
```

Open:

```text
Customer_Shopping_Behavior_Analysis.ipynb
```

### SQL Analysis

1. Create a database in PostgreSQL/MySQL/SQL Server.
2. Import the cleaned dataset.
3. Execute the SQL scripts provided in the project.

### Power BI Dashboard

1. Open the `.pbix` file in Power BI Desktop.
2. Refresh data connections if required.
3. Explore dashboard visuals and insights.
